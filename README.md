# Tech-Blush
Tech & Blush é um projeto desenvolvido como parte do desafio do Bootcamp Universia IA Generativa, em parceria com a DIO.
O objetivo deste repositório é apresentar um podcast criativo que explica conceitos de tecnologia utilizando linguagens, analogias e referências acessíveis ao universo feminino — tornando o aprendizado mais leve, acolhedor e divertido.
